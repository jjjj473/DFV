# JavaScript for Text Manipulation

element.textContent : Get or set the plain text of an element.
element.innerText : Read or update text with layout considerations.
document.createTextNode() : Create a standalone text node.
element.style.fontWeight = 'bold' : Make text bold.
element.style.fontStyle = 'italic' : Italicize text content.
element.style.textDecoration = 'underline' : Underline text.
element.style.color = 'red' : Change the text color.
element.style.fontSize = '1.2em' : Adjust the text size.
element.insertAdjacentText('beforeend', '...') : Insert text at a specific position.
element.replaceChildren() : Replace all child nodes, including text.
element.innerHTML += '<br>' : Insert a line break via HTML.
element.classList.toggle('highlight') : Toggle a class that changes text appearance.
document.execCommand('copy') : Attempt to copy selected text to the clipboard.
document.queryCommandSupported('copy') : Check if the copy command is allowed.
window.getSelection().toString() : Retrieve currently selected text.
